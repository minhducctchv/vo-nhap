# Watch props

Watch props thì phải dùng function

watch(() => props.myProp, (val) => {/*...*/})

# Watch nhiều biến

watch([bien1, bien2, bien3], (val) => {/* ... */});
watch({ ten1: bien1, ten2: bien2 }, (val) => {/* ... */})
